Read this before Proceeding

Requirements: Python 3.9, Google Chrome (latest)

Python 3.9 link:
https://www.python.org/ftp/python/3.9.11/python-3.9.11-amd64.exe

Python Installation link:
https://docs.python.org/3.9/using/windows.html#installation-steps

During installation click the checkbox "Add Python 3.9 to PATH"


After installing Python, Google Chrome
1. Unzip the archive file and double-click on the unzipped folder
2. Double-click on `env_setup.bat` file
3. Type `cmd` in address bar of the window


To run tests cases
4. Run `pytest` from the command prompt


After test cases completed report will be found in "Report" directory
